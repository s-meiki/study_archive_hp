## 1. 基本方針

当方は、個人情報の保護に関する法律その他の関係法令及びガイドラインを遵守し、個人情報を適切に取り扱います。

本サイトは、医療・薬学分野の教育アーカイブを扱うため、特に患者情報、登壇者情報、資料内の記載情報について、必要最小限・安全重視で運用します。

## 2. 取得又は取り扱う情報

当方は、次の情報を取得し、又は取り扱うことがあります。

### 2-1. 本サイトの閲覧に伴い生じる情報

- IP アドレス
- ブラウザ、OS、端末種別、言語設定等の技術情報
- 参照元 URL
- 閲覧日時、閲覧ページ、クリック情報

これらは、サーバ提供事業者、CDN、ホスティング事業者等により自動的に処理される場合があります。

### 2-2. 本サイト上に掲載するために取り扱う情報

- 登壇者の氏名、所属、役職、プロフィール、写真
- 勉強会タイトル、開催日、録画 URL、資料 URL
- 資料内に含まれる記載情報

当方は、患者情報、症例情報その他の機微情報について、公開に適さない情報を含まないよう確認します。

### 2-3. 問い合わせ対応に伴う情報

問い合わせフォームやメール窓口を設ける場合、氏名、メールアドレス、所属、問い合わせ内容その他当該対応に必要な情報を取得することがあります。

## 3. 利用目的

当方は、取得又は取り扱う情報を次の目的で利用します。

- 本サイトの運営、表示、保守、改善
- アーカイブ、資料、録画への導線提供
- 登壇者情報の掲載及び管理
- 著作権、個人情報保護、削除依頼その他の権利処理
- 不正アクセス、過剰アクセスその他のセキュリティ対策
- 問い合わせ、苦情、訂正、削除依頼への対応
- 法令順守及び事故対応

## 4. 個人情報の第三者提供

当方は、次の場合を除き、本人の同意なく個人情報を第三者に提供しません。

- 法令に基づく場合
- 人の生命、身体又は財産の保護のために必要がある場合
- 公衆衛生の向上又は児童の健全育成の推進のため特に必要がある場合
- 国の機関又は地方公共団体等に協力する必要がある場合
- そのほか法令上認められる場合

## 6. 外部サービスの利用と外部送信

本サイトでは、次の外部サービスを利用する場合があります。

### 6-1. GitHub Pages

本サイトは GitHub Pages を利用して公開される場合があります。その場合、アクセスに伴う通信情報やログ情報が GitHub により処理されることがあります。GitHub の取扱いは、GitHub の規約及びプライバシーステートメントを確認してください。

- GitHub Terms of Service
  - https://docs.github.com/en/site-policy/github-terms/github-terms-of-service
- GitHub Privacy Statement
  - https://docs.github.com/en/site-policy/privacy-policies/github-privacy-statement

### 6-2. YouTube 埋め込み

本サイトでは、動画再生のために YouTube 埋め込みプレーヤーを利用する場合があります。現在の設計では、動画はページ表示時には自動読込せず、利用者が `再生する` を押した時に読み込みます。

YouTube 埋め込みの読み込み時には、利用者に関する情報が Google 又は YouTube に送信される場合があります。送信されうる情報には、端末情報、ブラウザ情報、IP アドレス、閲覧ページ情報、再生操作情報等が含まれる場合があります。

送信先と目的:

- 送信先: Google LLC / YouTube
- 主な目的: 動画表示、再生機能の提供、不正利用防止、サービス改善、広告配信又は測定等

当方は、可能な範囲で `youtube-nocookie.com` を使用し、かつ遅延読込を行いますが、外部事業者による情報取扱いそのものを完全に制御するものではありません。

詳細は次の外部ポリシーを確認してください。

- YouTube Terms of Service
  - https://jp.youtube.com/t/terms
- YouTube Embedded Player information
  - https://support.google.com/youtube/answer/171780
- Google Privacy Policy
  - https://policies.google.com/privacy
